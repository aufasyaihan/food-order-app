<p align=center>A platform to add items to cart and send it to backend</p>

## Installation
### Clone the code
```
git init
git clone https://github.com/aufasyaihan/food-order-app.git
```
### Instal modules
```
npm install
```

## Run the program
```
npm run dev
```
